This folder contains all the example data used in this repository.
The descriptions of these data are listed below (all data are from the GAGE-seq):

- Bulk_HSC_100k.npy                 A pseudo-bulk data, at 100kb resolution, on chr1, from the HSC cell type.
- A10_A11.pairs                     A single-cell Hi-C data, in the `.pairs` form, on chr1.
- mBC_cell_ls.txt                   A filelist consisting of 100 single-cell Hi-C data, on chr1.
- chr1_data/                        This folder contains the 100 scHi-C data listed in `mBC_cell_ls`.
